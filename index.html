Anh's HTML Reference Guide:

This is the basic structure:

 
      <!DOCTYPE html> ---This is the instruction that tells the browser you are using HTML version 5
      <html>
        <head>
          <title>Website Title</title>
        </head>
        <body>
          content of website ...
        </body>
      </html>

Tags to use inside the <body> tag:

  • <p> ... </p> Use for new paragraphs, default 16px and top/bottom margins
  • <br> Inserts a line break and no closing tag needed
  • <div> ... </div> Used to format large blocks of HTML and stylesheets
  • <span> ... </span> Used for inline text styling
  • <a href="url" target="_new">some text that's clickable</a> Used to add a Link, target is the link location
  • <img src="url" alt="text"> Used to add an image

Lists:

  • <ol> ... </ol>Creates an numbered list
  • <ul> ... </ul> Creates an bulleted list
  • <li> ... </li> Used for each list item, it adds a number or symbol depending on the list type

Headlines <h1> ... <h6>:

  1. <h1> ... </h1>

    Creates the first largest headline

  2. <h2> ... </h2>

    Creates the second largest headline

  3. <h3> ... </h3>

    Creates the third largest headline

  4. <h4> ... </h4>

    Creates the fourth largest headline

  5. <h5> ... </h5>
    Creates the fifth largest headline
  6. <h6> ... </h6>
    Creates the sixth largest headline

Tables:

Element Tags What they do
<table> ... </table> Creates a table
<tr> ... </tr> Creates a row inside the table
<td> ... </td> Creates a cell in a row
<th> ... </th> Sets the table header

Frames:

<iframe src="url"> ... </iframe> Used to include another HTML page/website/video inside your HTML page.
